Pertanyaan yang Sering Diajukan (FAQ)

Apakah saya perlu memiliki pengetahuan coding untuk menggunakan template ini?

Untuk penggunaan dasar dan kustomisasi teks/gambar, tidak diperlukan pengetahuan coding yang mendalam. Namun, untuk perubahan struktur atau penambahan fitur kompleks, pemahaman dasar HTML, CSS, dan JavaScript akan sangat membantu.

Bagaimana cara menginstal atau meng-host template ini?

Template ini adalah file statis (HTML, CSS, JS) sehingga sangat mudah di-host. Anda bisa mengunggahnya ke layanan hosting statis seperti Netlify, Vercel, GitHub Pages, atau hosting web tradisional mana pun. Kami akan memberikan panduan dasar setelah pembelian.

Apakah ada dukungan teknis setelah pembelian?

Ya, kami menyediakan dukungan dasar via WhatsApp untuk lisensi Personal, dan dukungan prioritas/teknis untuk lisensi Developer dan Commercial. Detail dukungan akan dijelaskan saat pembelian.

Bisakah saya menggunakan template ini untuk beberapa proyek?

Tergantung pada jenis lisensi yang Anda beli. Lisensi Personal hanya untuk 1 proyek personal. Lisensi Developer memungkinkan hingga 3 proyek komersial. Lisensi Commercial memberikan hak penggunaan tak terbatas untuk berbagai proyek dan klien.
